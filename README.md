CineMundo Project:
Project Overview:
 - CineMundo is a comprehensive web platform for selling movie tickets, developed as a school project. The application is designed to be intuitive and functional, allowing users to view showtimes, get detailed information about films, and securely purchase tickets.

The project is divided into two main areas:

 - Commercial: Focused on the user experience and the purchasing process.
 - Administrative: Internal management of data, sales, and users.
 - Detailed Features
 - Commercial Area (Frontend)
 - Film Showcase: Display of movies currently in theaters with complete information (synopsis, cast, director, genre, duration, and classification).
 - Showtime Search: Allows users to find movie sessions by date and cinema location.
 - User Registration: A registration system so customers can manage their purchases.
 - Ticket Purchase: A process for selecting and buying tickets with various payment options.
 - Email Confirmation: After a purchase, an email is sent to the user with all the order details.
 - Administrative Area (Backend)
 - Registration Management: Administration of films, sessions, and cinemas.
 - Administrative Reports: Generation of reports on sales, registered users, and platform performance.

Technologies Used:
 - The project's architecture was planned to be modern and scalable, using the following technologies:

Frontend:

 - React: A JavaScript library for building the user interface, ensuring a dynamic and reactive experience.
 - HTML5/CSS3: The structure and styling of the application.

Backend:
 - Node.js: A JavaScript runtime environment for the server, responsible for business logic and APIs.

Database:
 - MySQL: A relational database to store and manage information about films, users, sessions, and sales.
